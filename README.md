### Steps

1. **Clone and Install**

   ```bash

   cd react-template
   bun install
   ```

2. **Run Development Server**

   ```bash
   bun run dev
   ```

3. **Build for Production**

   ```bash
   bun run build
   ```

4. **Preview Production Build**

   ```bash
   bun run preview
   ```
